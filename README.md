## **📥 Installation & Setup**
1️⃣ **Clone the repository**  
```sh
git clone https://github.com/your-username/school-website.git
cd school-website
```

2️⃣ **Install dependencies**  
```sh
npm install
```

3️⃣ **Start the development server**  
```sh
npm run dev
```
> Open `http://localhost:3000/` in your browser.
